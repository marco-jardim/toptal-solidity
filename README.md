Requirements:
- node.js ^16.0.0
- solidity ^0.8.9

Install dependencies:
```npm i```

Compile:
```npx hardhat compile```

Testing:
```npx hardhat test```

Deploy:
```npx hardhat run scripts/deploy.js --network rinkeby```

Contract verification for Etherscan Web3 interaction & transparency:
```npx hardhat verify --network rinkeby CONTRACT_ADDRESS```
